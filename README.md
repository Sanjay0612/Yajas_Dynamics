# Yajas Dynamic - Professional Industrial Valves Manufacturer Website

## Website Overview

This is a complete, professional multipage website for Yajas Dynamic, an industrial valves manufacturer based in Bangalore, India.

## Website Structure

### Pages Included:
1. **index.html** - Home page with featured products and company introduction
2. **about.html** - Company information, mission, values, and certifications
3. **products.html** - Main products catalog with all valve types
4. **products-ball.html** - Ball Valves detail page with specifications
5. **products-gate.html** - Gate Valves detail page
6. **products-globe.html** - Globe Valves detail page
7. **products-check.html** - Check Valves detail page
8. **products-butterfly.html** - Butterfly Valves detail page
9. **products-plug.html** - Plug Valves detail page
10. **applications.html** - Industry-specific applications 
11. **blog.html** - Blog article listing page
12. **blog-1.html to blog-8.html** - Individual blog article pages
13. **certifications.html** - All certifications and standards
14. **contact.html** - Contact form and business information

### Supporting Files:
- **styles.css** - Complete stylesheet with responsive design
- **logo.jpg** - Company logo 
- **favicon.jpg** - Browser favicon 
- **images/** - Directory for product images

## Setup Instructions

### 1. File Organization
Create the following directory structure:

```
yajas-dynamic/
├── index.html
├── about.html
├── products.html
├── products-ball.html
├── products-gate.html
├── products-globe.html
├── products-check.html
├── products-butterfly.html
├── products-plug.html
├── applications.html
├── blog.html
├── blog-1.html to blog-8.html
├── certifications.html
├── contact.html
├── styles.css
├── logo.jpg
├── favicon.jpg
└── images/
    ├── ball-valve.png
    ├── gate-valve.png
    ├── globe-valve.png
    ├── check-valve.png
    ├── butterfly-valve.png
    └── plug-valve.png
```

### 2. Add Your Assets
- Replace `logo.jpg` with your company logo
- Replace `favicon.jpg` with your favicon
- Add product images to the `images/` folder:
  - ball-valve.png
  - gate-valve.png
  - globe-valve.png
  - check-valve.png
  - butterfly-valve.png
  - plug-valve.png

### 3. Update Contact Information
Open each HTML file and update:
- Phone number: +91-XXXX-XXXX-XX
- Email: info@yayinautomation.com (already correct)
- Business address and hours

### 4. Launch the Website
Option A: Local Testing
- Double-click any HTML file to open in your browser

Option B: Live Server (Recommended)
- Use VS Code with Live Server extension
- Open project folder in VS Code
- Right-click index.html and select "Open with Live Server"

Option C: Deploy Online
- Upload all files to a web hosting service
- Use static hosting like Netlify, GitHub Pages, or Vercel

## Customization

### Change Colors
Edit `styles.css` and replace color codes:
- Primary color: #1b5e4f (dark teal)
- Secondary color: #2a8b7a (lighter teal)
- Text color: #333 (dark gray)

### Add More Products
- Create new files: `products-[valve-type].html`
- Add links to `products.html`
- Create corresponding images

### Expand Blog
- Create new files: `blog-[number].html`
- Add links to `blog.html`
- Update publication dates

### Update Contact Information
Search for "+91-XXXX-XXXX-XX" and replace with actual phone number
Search for "Bangalore, Karnataka, India" and update address as needed
